# STRATEGIE UI/UX - MEGA-PLATEFORME "BUSINESS IN A BOX"

## Document de Reference - Mars 2026

### Vue d'ensemble

Ce document definit la strategie UI/UX complete pour une mega-plateforme integree regroupant 20+ modules business (E-commerce, CRM, Comptabilite, Inventaire, Telephonie, Contenu, Marketing, Support, Projets, AI Copilots) en un seul ecosysteme unifie.

### Decisions Architecturales Cles

1. **Navigation** : Sidebar Spaces (inspire de Zoho One) + Command Palette Cmd+K (inspire de Linear/Zoho QuickNav)
2. **Design System** : Tokens semantiques 3 niveaux (inspire IBM Carbon + Salesforce SLDS 2)
3. **Palette** : Transformative Teal primary + Indigo secondary (tendance WGSN/Coloro 2026)
4. **Themes** : Light + Dark + 3 modes densite (Compact/Standard/Focus)
5. **Record Page** : Pattern HubSpot (header + sidebar proprietes + timeline + associations)
6. **Dashboards** : Personnalisables par role avec widgets drag-and-drop (inspire Zoho Dashboard 2.0)
7. **AI** : Copilot contextuel integre dans chaque module (inspire Microsoft Copilot + HubSpot Breeze)
